Welcome to the Movie API Documentation

The MyFlix API allows users to access information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.

Registered users can sign up, update their profiles, and maintain a personalized list of favorite movies.

API Endpoints

Request URL Method Request Body (if applicable) Response Description
Get all movies /movies GET None List of all movies Returns an array of all movies
Get movie by title /movies/:title GET None Movie object Returns data about a single movie by title
Get genre by name /genres/:name GET None Genre description Returns description of a genre by name
Get movies by genre and director /movies?genre=Comedy&director=Wes%20Anderson GET None Filtered movie list Returns movies that match both genre and director
Get director by name /directors/:name GET None Director details Returns bio, birth year, and death year of a director
Register new user /users POST { "username": "user1", "password": "123", "email": "user@example.com" } Confirmation message Allows a new user to register
Update user info /users/:username PUT { "username": "newname", "email": "new@example.com" } Updated user data Allows a user to update their info
Update movie info /movies/:title PUT { "Genre.Description": "Updated description" } Updated movie object Updates details of a specific movie
Add movie to favorites /users/:username/favorites/:movieTitle POST None Confirmation message Adds a movie to a user's favorite list
Remove movie from favorites /users/:username/favorites/:movieTitle DELETE None Confirmation message Removes a movie from a user's favorite list
Deregister user /users/:username DELETE None Confirmation message Removes a user account
